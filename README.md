# State-of-data-brazil-2023
Este repositório abriga o projeto acadêmico da disciplina de Tópicos em Big Data em Python. O projeto analisa os dados da pesquisa anual "State of Data Brazil", realizada pela comunidade Data Hackers em parceria com a Bain & Company, que investiga o panorama do mercado de trabalho em dados no Brasil.
